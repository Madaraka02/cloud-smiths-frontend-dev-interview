# Dog Breeds Viewer App 

A React application that allows users to browse dog breeds, view random images, and save favourite images.


## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Madaraka02/cloud-smiths-frontend-dev-interview.git
cd cloud-smiths-frontend-dev-interview
```


### 2. Install dependencies

```bash
npm install
```


### 3. Configure environment variables

Create a `.env` file in the root of the project:

```env
VITE_AUTH_API_URL=https://dummyjson.com/auth
VITE_DOGS_API_URL=https://dog.ceo/api
```

### Environment Variables Explained

* `VITE_AUTH_API_URL` → Authentication API used for login
* `VITE_DOGS_API_URL` → Public Dog API used to fetch breeds and images


## Running the App

### Start development server

```bash
npm run dev
```

The app will be available at:
[http://localhost:5173](http://localhost:5173)


### Run tests

```bash
npm run test
```
